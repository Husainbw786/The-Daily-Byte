Given a binary tree, return a list of strings containing all root to leaf paths.

Ex: Given the following tree…    

       1
     /   \
    2     3
return ["1->2", "1->3"]     
Ex: Given the following tree…    

        8
       / \
      2  29
        /  \
       3    9    
return ["8->2", "8->29->3", "8->29->9"]
